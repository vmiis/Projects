
How to run a web server for this micro web application

1. Install Python if haven't (https://www.python.org/downloads/)
2. Replace start-server.123 to start-server.bat if haven't (files with .bat extension are not allowed to be sent with email)
3. Run start-server.bat, you will see a port number (a random number), for example the number may be 57193
4. Use web browser to see this micro web application, http://localhost:57193
