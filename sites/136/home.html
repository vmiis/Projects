This micro web application contains the following files

Clinet Side
  • index.html layout for header, contents and footer
  • header.htm logo, menu, login/logout etc
  • footer.htm copyright info, links etc
  • home.htm landing page, show some information
  • calendar-month.html monthly canlandar
  • calendar-week.html weekly canlandar
  • calendar-day.html daily canlandar
  • data-booking.html view and search data in a cloud database table
  • form-booking.html data collection form
  • form-booking-2.html add or edit a record (with pre-filled with data)
  • data-car.html view and search data in a cloud database table
  • form-car.html data collection form
Server Side
  • server.js providing web hosting services
  • start-server.bat start server


Download and run this micro web app on a local computer
